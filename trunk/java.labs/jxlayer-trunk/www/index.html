JXLayer project

Welcome to the home of the JXLayer, the universal decorator for Swing components.

The JXLayer project is available under BSD licence and is free for commercial or non-commercial use.

With help of JXLayer you can easily decorate your compound components and catch all Mouse, Keyboard and FocusEvent for all its subcomponents.
JXLayer is very friendly to your applications, it uses only public Swing API and doesn't change any global settings like RepaintManager, EventQueue or frame's glassPane.

JXLayer is extremely compatible with any other components or libraries and can be used in restricted environments, like unsigned applets.

  • Don't like verbose AWTEventListeners?
  • Got tired of using frame's glassPane?
  • Going to write another custom RepaintManager, but don't want to break compatibility with one from SwingX?
Wrap your component with JXLayer and you'll have the complete control over it and all its subcomponents.

Download

The sources can be downloaded from the Subversion repository.
Jar and zip files are available for Maven and also for direct download:
  • jxlayer.jar - the binaries of JXLayer and its built-in extentions.
  • jxlayer-demo.jar - the binaries of JXLayer's demos (you should also have filters.jar and swingx.jar to run them)
  • jxlayer-src.zip - zipped source files of JXLayer and all its demos.
  • jxlayer-doc.zip - complete JXLayer's javadoc
  • jxlayer-all.zip - everything mentioned above, zipped in one file.
