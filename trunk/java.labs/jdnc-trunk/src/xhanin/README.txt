JExplose - JDNC Incubator

Overview
========
Enhance your end-user MDI experience with JExplose! Extremely simple to integrate with your existing Swing
MDI products (using Swing JDesktopPane or any custom subclass), JExplose allows your end-users to easily
navigate between their JInternalFrames.

Installation
============
include the jexplose.jar delivered in your classpath.

Integration
===========
see doc/overview.html

Demo
====
1) build using ant
2) run java -jar build/jexplose-demo.jar
